It contains classes which defines the inner component for genetic algorithm and other component which can be otherwise plugged in.

Package contains classes:
- EnvSelectAnnot is the annotation for environmental selectors
- FitnessAnnot is the annotation for fitness functions
- GenAnnot is the annotation for generators
- MateSelectAnnot is the annotation for mating selectors
- MOperatorAnnot is the annotation for mutation operators
- XOperatorAnnot is the annotation for crossover operators
- PopInitAnnot is the annotation for population initializators
- PopulationAnnot is the annotation for populations from genetic algorithm
