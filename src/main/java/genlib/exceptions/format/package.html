It contains package and classes which defines the new string format exceptions for better handling of possible errors

Package contains classes:
- FitCompStringFormatException is the exception thrown when format of the fit comparator component is incorrect
- FitnessStringFormatException is the exception thrown when format of the fitness function component is incorrect
- OperatorStringFormatException is the exception thrown when format of the operator component is incorrect
- PopulationInitStringFormatException is the exception thrown when format of the population init component is incorrect
- PopulationTypeStringFormatException is the exception thrown when format of the population type is incorrect
- SelectorStringFormatException is the exception thrown when format of the selector component is incorrect
