It contains interface and classes with generator of starting individuals.

Package contains interface:
- Generator is the main interface that all the generator must implement - TreeGenerator is the submain interface which extends Generator and that all tree generators must implement.

Package contains classes:
- WekaJ48TreeGenerator is the tree generator that generates the trees with J48 classifier
- WekaSimpleStumpGenerator is the tree generator that generates the decision stumps
- WekaTreeGenerator is the tree generator that generates the trees with some classifier from Weka with specific format
- DummyTreeGenerator is the test generator that generates the same testing tree
