It contains interfaces, packages and classes used with classifier objects that make the models with genetic algorithms.

Package contains interface:
- Classifier is the interface for all kinds of classifiers

Package contains packages:
- classifierextension is the package with extensions for weka and non weka classifiers
- common is the package that contains common procedures and functions that can be used by weka and non weka classifiers
- origin is the package with non weka classifiers
- weka is the package with weka classifiers
