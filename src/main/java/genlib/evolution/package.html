It contains packages, interfaces and classes that are in some way used in evolution algorithm

Package contains packages:
- fitness is the package with all the fitness functions and comparators used in EA
- individuals is the package with container for each individual
- operators is the package with all the operators (mutation and crossover) used in EA
- population is the package with container for population of individuals
- selectors is the package with all the selectors (mating and environmental)

Package contains classes:
- EvolutionAlgorithm is the main class that execute the steps of evolution algorithm. It uses all the component (selectors, operators, fitness functions) from this package and creates the new individuals.
