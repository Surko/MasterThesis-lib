It contains packaged, interfaces and classes to define fitness functions.

Package contains packages:
- comparators is the package which contains all the comparators that can used to sort the individuals in population
- tree is the package that contains fitness functions for trees

Package contains interface:
- FitnessFunction is the basic interface that must be somewhat implemented with all kinds of functions

Package contains class:
- TestFit is the testing fitness function that returns complex fitness (even if it's not initialized yet)
