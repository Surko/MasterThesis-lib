It contains interface and classes define the structure of tree

Package contains inteface:
- Node is the interface for all kind of nodes

Package contains classes:
- BinaryNode is the class for the binary nodes
- MultiWayNode is the class for the non binary nodes
- BinaryHeightNode is the class for the binary nodes which computes their height
- MultiWayHeightNode is the class for the non binary nodes which computes their height
