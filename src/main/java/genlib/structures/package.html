It contains packages, interfaces and classes that in some way defines or extend some kind of structure.

Package contains packages:
- data is the package that interfaces in case of future extension to make the application stand alone without weka
- extensions is the package that contains extensions for structures (nodes)
- trees is the package that defines the structure of individuals (all the types of nodes)

Package contains class:
- Data is the class that serves the purpose of container for GenLibInstances or Instances
